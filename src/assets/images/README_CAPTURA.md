# Imagens do Componente Captura

## Instruções

Coloque as imagens dos painéis do componente Captura na pasta `src/assets/images/`:

1. **akani-uembi.png** - Imagem para o painel "Akani UEMBY"
   - Tamanho recomendado: 200x200px ou maior (proporção 1:1)
   - Formato: PNG com fundo transparente (preferencial) ou JPG
   - Tema: Hardware dedicado para captura ao vivo

2. **akani-apyk.png** - Imagem para o painel "Akani APYK"
   - Tamanho recomendado: 200x200px ou maior (proporção 1:1)
   - Formato: PNG com fundo transparente (preferencial) ou JPG
   - Tema: Transferência de arquivos individual ou em bloco

3. **akani-ayvu.png** - Imagem para o painel "Akani AYVU"
   - Tamanho recomendado: 200x200px ou maior (proporção 1:1)
   - Formato: PNG com fundo transparente (preferencial) ou JPG
   - Tema: Captura de áudio

## Localização dos arquivos:
- `src/assets/images/akani-uembi.png`
- `src/assets/images/akani-apyk.png`
- `src/assets/images/akani-ayvu.png`

As imagens serão exibidas automaticamente no componente quando os arquivos forem adicionados.
