# Imagens do Carrossel

## Instruções

Coloque as imagens dos painéis do carrossel na pasta `src/assets/images/`:

1. **captura.png** - Imagem para o painel "Captura"
   - Tamanho recomendado: 300x300px ou maior (proporção 1:1)
   - Formato: PNG com fundo transparente (preferencial) ou JPG
   - Tema: Equipamentos ou interfaces de captura de conteúdo

2. **playout.png** - Imagem para o painel "Playout"
   - Tamanho recomendado: 800x600px ou maior (proporção 4:3 ou 16:9)
   - Formato: JPG (recomendado para fotos) ou PNG
   - Tema: Sala de controle de broadcast com monitores e operadores
   - Descrição: Imagem de uma sala de controle de broadcast moderna com múltiplos monitores, painéis de controle e operadores trabalhando

3. **gerenciamento-conteudo.png** - Imagem para o painel "Gerenciamento de Conteúdo"
   - Tamanho recomendado: 300x300px ou maior (proporção 1:1)
   - Formato: PNG com fundo transparente (preferencial) ou JPG
   - Tema: Plataforma de gerenciamento de conteúdo de mídia

## Localização dos arquivos:
- `src/assets/images/captura.png`
- `src/assets/images/playout.png`
- `src/assets/images/gerenciamento-conteudo.png`

As imagens serão exibidas automaticamente no carrossel quando os arquivos forem adicionados.
